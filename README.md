# BioSafe

BioSafe es una plataforma web prototipo orientada a la gestión de vacunación materno-infantil, validación mediante código QR y alertas epidemiológicas preventivas con apoyo de inteligencia artificial.

## Descripción

El proyecto busca modernizar el seguimiento de esquemas de inmunización mediante una interfaz web intuitiva, pensada para centros de salud, personal médico y administradores. Actualmente funciona como prototipo visual y funcional estático.

## Funcionalidades principales

- Página informativa del sistema BioSafe.
- Panel administrativo estático.
- Visualización de indicadores de vacunación.
- Sección de alertas epidemiológicas preventivas.
- Validación QR simulada.
- Diseño responsive con React y Tailwind CSS.

## Tecnologías utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React


### Arquitectura General:

- B2B SaaS (Software as a Service) con arquitectura Multi-tenant (Multi-inquilino) .
- API REST (Cliente-Servidor) y Microservicios lógicos.

### Backend & Base de Datos:

-Supabase PostgreSQL
